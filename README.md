# FiatCurriculux

Keep your CV up to date on all your socials with minimal reformatting effort.

All CV contents are kept in the `cv_contents.json`. Run the `make_xxx.py` script to generate an up to date CV in the desired format.

## Features :
- Flexible structure as described by `cv_contents.json`
- CV items with (and/or) :
    - Description text
    - Item list
    - Summary table
- url parsing for :
    - hal.science articles
    - doi documents
